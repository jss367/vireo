Import

Bring your photos into Spotter. Scan a folder on disk, or import keywords from a Lightroom catalog.

Scan a Photos Folder

Point Spotter at the root of your photos folder. It will walk all subfolders, discover image files (JPEG, RAW, TIFF, etc.), read EXIF timestamps, import any existing XMP sidecar keywords, and generate thumbnails for browsing.

Generate Thumbnails

Create a local thumbnail cache so you can browse photos even when the NAS is offline. Only generates thumbnails for photos that don't already have one.

Import from Lightroom Catalog

If your keywords live in Lightroom .lrcat catalogs (and haven't been written to XMP sidecars), import them here. This reads keywords from the catalog database and adds them to Spotter. You can optionally write them to XMP sidecars too.
